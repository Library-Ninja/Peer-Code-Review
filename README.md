# Peer-Code-Review
This Java program takes a plaintext message as user input, then shifts each letter of the plaintext by the next number in the Fibonacci sequence. For example, the plaintext “aaaaaa” would become “abbcdf” because each letter is shifted by one of the first six digits of the Fibonacci sequence: 0, 1, 1, 2, 3, 5. (a + 0 = a, a + 5 = f, etc.) The ciphertext output is then printed.
